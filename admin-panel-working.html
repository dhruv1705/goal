Goals App Admin Panel (Working Solution)

Complete admin access with service role configuration

🔧 Admin Configuration Required

To access all data, you need to provide your Supabase service_role key (not the anon key). This bypasses RLS and gives full admin access.

How to get Service Role Key:
1. Go to your Supabase project dashboard
2. Navigate to Settings → API
3. Copy the "service_role" key (not the "anon" key)
4. Paste it above and click "Connect with Service Role"
